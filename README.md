# AI Website Analysis Chatbot

## Overview

This project is an AI-powered chatbot developed during my internship at Quantal AI. The application leverages Large Language Models (LLMs) to analyze website content, answer user queries, and provide intelligent recommendations for improving website quality and user experience.

The project demonstrates practical implementation of modern AI frameworks for conversational applications.

---

## Features

* Website content analysis
* Intelligent question answering
* Context-aware responses using LLMs
* Prompt engineering techniques
* Interactive chat interface
* Modular architecture
* MCP-based integrations for extending AI capabilities

---

## Tech Stack

* Python
* LangChain
* Streamlit
* Together AI API
* MCP (Model Context Protocol)
* Git & GitHub

---

## Workflow

```
User Query
```
